Quick Tips

  • Tap this icon to go to the Global menu, where you can:
    • Search using SpeedSearch.
    • Use Quick Actions to quickly add an Account/Contact.
    • Use Go To to access all of the record types in the Mobile Client.
    • Expand Other to access the following features:
      • Use Configure Menu to change the items in the Go To menu and specify the order in which they appear. The first item in the list on the Configure Menu screen becomes your default view and loads first after you log on.
      • Use Settings to clear saved credentials, clear storage, access error logs, empty their Briefcase by clearing Offline Options, set the Mobile Client to use the 12 hour clock instead of the 24 hour clock, and set a new language for the Mobile Client.
      • Find Help.
      • Sign Off.
      • Connection shows whether your Mobile Client is Online (the header bar in records will display as blue) or Offline (the header bar will display as gray).
  • Tap this icon to go to the Contextual (right) menu and View Settings. Options available vary by record type.
    • See Available hashtags
    • View KPIs (Key Performance Indicators)
    • Expand Groups, Configure to select the groups to show on your list views. This is available on Account, Contact, Lead, Opportunity, and Ticket list views.
    • Select from Layouts. This is available on Account, Contact, Lead, Opportunity, and Ticket list views. The layout you select becomes the default when you return to this record type later.
      • Summary layout shows as 4 fields in 1 column. This is the default.
      • Detail layout shows as 9 fields in 1 column.
        Note: If you have access to the Infor CRM SLX Web or Windows Client, you can modify how some fields display in these Mobile Summary and Detail layouts.
        • From the Web Client Account, Contact, Lead, Opportunity, or Ticket List view, right-click on a group tab, click Edit, and then click the Layout tab:
        • Drag and drop columns to change their order.
        • Display currency fields in green text: select the Potential column, click Edit, expand the Format type field drop-down and select Currency, click OK.
        • Display phone numbers as a clickable link: select the Phone number click Edit expand the Format type field drop-down and select Phone number, click OK.
        See the Web or Windows Client help for more information about editing groups.
  • This icon indicates that you can expand and collapse panels to show/hide additional details.
  • This icon indicates that you can add a new record to Infor CRM SLX. It is available on all list views.
  • When the Refresh button is available, tap it to immediately refresh information. Select a record to limit the refresh to that record. When no record is selected, data for the entire group is refreshed.
  • Blue text in detail views signifies a link action (such as browsing to a URL or jumping to an Account).
  • Note: When using iPhone or iPad, and Infor CRM SLX Mobile is accessed from a shortcut on the home screen, the Map link opens in a new view and the Web link is disabled.

Quick Actions

  • In list views, tap the Ellipsis icon to see and access short-cuts to frequently-used features. The options available will vary depending on the record type. (Some quick action icons may be unavailable. For example, in My Schedule, the Decline action will be unavailable if you have already accepted the meeting.)
  • Use Other>Configure Menu to change the visibility and order of the items in the Quick Action pane.

Working Offline

    Features allow read-only access to select records and entities when your Mobile Client is offline or loses connectivity.
  • Recently Viewed shows any detail views visited, so they are available if connectivity is lost. Recently Viewed is available on the Global Menu.
  • My Briefcase allows you to select records to be available when offline. To add items to My Briefcase, in the detail view, open a record, and then click the briefcase icon located in the header bar. All related data for that record is also moved to the Briefcase. When you open a record from the My Briefcase view, the header bar is gray because the Briefcase is always in offline mode.
  • My Schedule shows today's activities. My Schedule is available on the Global Menu. You can move all activities in the My Schedule list view into My Briefcase. To do this, open the My Schedule list view and click the briefcase icon located in the header bar.
  • Notes created while offline synchronize the next time you connect to Infor CRM SLX.

Search

  • There are multiple ways to search:

    • From the Global menu, find all related accounts, contacts, leads, opportunities, activities, notes/history, and tickets. Enter more text to improve the accuracy of your search.
