Infor CRM Back Office Extension 簡介

Infor CRM Back Office Extension (ICBOE) 將下列實體新增至行動用戶端:
  • 發票
  • 報價
  • 銷售訂單
  • 貨運
  • 應收帳款
若要查看這些實體，您必須啟用檢視:
  • 依序點選全域功能表 > 其他 > 設定功能表，點選發票、應收帳款和/或貨運。在頂端功能表列中，點選勾選記號圖示以接受選擇並且退出設定功能表。
  • 如果需要，設定實體的預設群組: 在實體清單檢視中，點選未設定預設群組。點選此處以設定群組並且選擇查看該實體的所有可用記錄或是只查看該實體中您的記錄。

  • 實體詳細資料檢視相關項目標籤提供存取其他資料。開啟下列實體類型的詳細資料檢視以存取相關項目標籤中的項目:

    • 發票: 發票項目與應收帳款項目。
    • 報價: 報價行、附件、銷售人員與同步歷程記錄。
    • 應收帳款: 應收帳款項目
    • 銷售訂單: 訂單行、發票行、貨運行、附件和同步歷程記錄
    • 貨運: 貨運行

快速提示

  • 可用於報價和銷售訂單的群組。
  • 「報價」與「訂單」行項目中可以使用刪除按鈕，點選此按鈕可刪除 CRM 中的目前記錄。此按鈕只能在行項目的詳細資料檢視中使用。
  • 離線工作最近檢視和我的公事包功能可以在 ICBOE 記錄中使用。

儀表板

使用儀表板，以清單小組件形式查看客戶相關的可自訂、預定義分析。

在客戶詳細資料配置中，點選儀表板標籤。

  • 財務快照: 以可設定的 30、60、90、180 與 365 天的期間顯示資訊。
    • 已開發票收益: 與過去 x 天中建立之客戶相關發票的合計總金額。
    • 已開發票利潤:(合計總金額) 減去與過去 x 天中建立之客戶相關的發票 (合計成本)。
    • 已開發票利潤率：(已開發票利潤金額) 除以與過去 x 天中建立之客戶相關的發票的 (總金額)。
    • 收益年同比增長率: (過去一年的收益減去 2 年前及 1 年前之間的收益) 除以 ( 2 年前及 1 年前之間的收益)。
    • 利潤年同比增長: (合計總金額) 減去與過去 x 天中建立之客戶相關發票的 (合計成本)。
    • 利潤率年同比增長率: 與過去 x 天中建立之客戶相關發票的 (過去一年的利潤減去 2 年前及 1 年前之間的利潤的差) 除以 (2 年前及 1 年前之間的利潤)。
  • 新增交易: 顯示可設定的 7、14、30 以及 90 天期間 (包括目前日期)。會顯示發票數目及其總值。
    • 新增報價: 與過去 x 天中建立之客戶相關的報價數目。
    • 新增訂單: 與過去 x 天中建立之客戶相關的訂單數目。
    • 新增貨運: 與過去 x 天中建立之客戶相關的貨運總目。
    • 新增發票： 與過去 x 天中建立之客戶相關的發票數目。
  • 未結交易: 顯示目前日期之前可設定的 0+、30+、60+、90+ 以及 180+ 天期間的未結時效報價、時效發票以及時效銷售訂單。將會顯示交易記錄數目及其總值。
    • 時效報價: 與過去 x 天中建立之客戶相關的未結報價總數。
    • 時效發票: 與過去 x 天中建立之客戶相關的未結發票總數。
    • 時效銷售訂單: 與過去 x 天中建立之客戶相關的未結銷售訂單總數。

ICBOE 搜尋欄位

針對每個實體，在查找列使用下方指定的欄位篩選清單檢視結果。

    搜尋欄位

    搜尋欄位
