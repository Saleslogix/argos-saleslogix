Presentazione di Infor CRM Back Office Extension

Infor CRM Back Office Extension (ICBOE) aggiunge le entità seguenti al client mobile:
  • Fatture
  • Offerte
  • Ordini di vendita
  • Spedizioni
  • Crediti
Per vedere queste entità, è necessario abilitare le viste:
  • Fare clic sul menu Globale > Altro > Configura menu, quindi toccare Fatture, Crediti e/o Spedizioni. Nella barra dei menu nella parte superiore fare clic sull’icona del segno di spunta per accettare le selezioni e uscire dalla schermata Configura menu.
  • Se necessario, impostare un gruppo predefinito per un’entità. Nella vista elenco dell’entità fare clic su Nessun gruppo predefinito impostato. Fare clic qui per configurare i gruppi. Scegliere quindi se si desidera vedere tutti i record disponibili per tale entità o solo i propri record.

  • Le schede Elementi correlati delle viste dettagli delle entità consentono di accedere a informazioni aggiuntive. Aprire una vista dettagli per i seguenti tipi di entità per accedere agli elementi elencati nella scheda Elementi correlati:

    • Fatture: Articoli fattura e Voci di credito.
    • Offerte: Righe offerta, Allegati, Venditori e Cronologia di sincronizzazione.
    • Crediti: Voci di credito
    • Ordini di vendita: Righe ordine, Righe fattura, Righe spedizione, Allegati e Cronologia di sincronizzazione
    • Spedizioni: Righe spedizione

Suggerimenti rapidi

  • I gruppi sono disponibili per offerte e ordini di vendita.
  • Il pulsante Elimina è disponibile per le voci Offerta e Ordine. Toccare il pulsante per eliminare il record corrente da CRM. Questo pulsante è disponibile solo nella vista dettagli della voce.
  • Le funzionalità Visualizzati di recente e Sincronia file della modalità offline sono entrambe disponibili per i record ICBOE.

Dashboard

Usare i dashboard per vedere le analisi predefinite e personalizzabili relative all’azienda sotto forma di widget elenco.

Nel layout dei dettagli sull’azienda, fare clic sulla scheda Dashboard

  • Snapshot finanziario: visualizza le informazioni relative a un periodo di tempo configurabile di 30, 60, 90, 180 e 365 giorni.
    • Ricavi fatturati: importo complessivo totale delle fatture relative all’azienda create negli ultimi x giorni.
    • Profitto fatturato: l’importo complessivo totale meno il costo totale delle fatture relative all’azienda create negli ultimi x giorni.
    • Margine fatturato: l’importo del profitto fatturato diviso per l’importo complessivo delle fatture relative all’azienda create negli ultimi x giorni.
    • Ricavi anno su anno: i ricavi dell’anno precedente meno i ricavi di 1 o 2 anni precedenti divisi per i ricavi di 1 anno o 2 anni precedenti.
    • Profitto anno su anno: l’importo complessivo totale meno il costo totale delle fatture relative all’azienda create negli ultimi x giorni.
    • Margine anno su anno: il margine dell’anno precedente meno il margine di 1 o 2 anni precedenti diviso per il margine di 1 o 2 anni precedenti delle fatture relative all’azienda create negli ultimi x giorni.
  • Nuove transazioni: visualizza le transazioni relative a un periodo configurabile di 7, 14, 30 e 90 giorni compresa la data corrente. Visualizza sia il numero delle fatture che il loro valore totale.
    • Nuove offerte: il numero delle offerte relative all’azienda create negli ultimi x giorni.
    • Nuovi ordini: il numero di ordini relativi all’azienda creati negli ultimi x giorni.
    • Nuove spedizioni: il numero di spedizioni relative all’azienda create negli ultimi x giorni.
    • Nuove fatture: il numero di fatture relative all’azienda create negli ultimi x giorni.
  • Transazioni aperte: visualizza le offerte, le fatture e gli ordini di vendita aperti suddivisi per anzianità in base a un periodo configurabile di oltre 0, 30, 60, 90 e 180 giorni dalla data corrente. Sono riportati sia il numero di record che il valore totale delle transazioni.
    • Offerte per anzianità: il numero totale di offerte aperte relative all’azienda create negli ultimi x giorni.
    • Fatture per anzianità: il numero totale di fatture aperte relative all’azienda create negli ultimi x giorni.
    • Ordini di vendita per anzianità: il numero totale di ordini di vendita aperti relativi a questa azienda creati negli ultimi x giorni.
